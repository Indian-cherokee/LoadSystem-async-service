# Асинхронный для расчёта нагрузки на междуэтажное перекрытие
### Никифоров А.Ю. ИУ5-53Б
В данном репозитории представлен асинхронный веб-сервис для расчёта нагрузки на междуэтажное перекрытие.

Актуальная ветка - async-service

Навигация по проекту:

- Фронтенд - https://github.com/Indian-cherokee/LoadSystem-frontend
- Основной веб-сервис - https://github.com/Indian-cherokee/LoadSystem-backend
- Асинхронный веб-сервис - https://github.com/Indian-cherokee/LoadSystem-async-service
